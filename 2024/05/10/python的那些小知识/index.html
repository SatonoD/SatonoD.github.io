Python的基本数据结构

  1. 列表（List）
    列表是一种有序的集合，可以保存各种类型的数据，并且支持增加、删除和其他多种操作。列表中的元素是可变的，可以通过索引访问。

  2. 元组（Tuple）
    元组与列表类似，也是一种有序的集合，但元组是不可变的。一旦创建了元组，就不能修改其内容。元组通常用于保护数据不被修改。

  3. 字典（Dictionary）
    字典是一种键值对集合，非常适合快速检索数据。字典中的键必须是唯一的，并且是不可变类型，如字符串或元组。

  4. 集合（Set）
    集合是一个无序的、不包含重复元素的集合类型。它主要用于测试成员资格、删除重复项和进行数学运算如并集、交集、差集等。

这些数据结构各有特点，适用于不同的场景：
列表：适合需要经常修改数据的情况，如添加或删除元素。
元组：适合创建不应更改的数据集，如函数返回多个值。
字典：适合需要关联数据的情况，非常高效地通过键检索值。
集合：适合需要进行集合操作以及需要唯一元素的情况。

numpy.array与list的区别/优势

  1. 性能
    速度：numpy 数组在执行操作时通常比 Python 列表快得多。这是因为 numpy 内部使用优化过的 C 和 Fortran 代码来执行数学运算，而 Python 列表则是由 Python 语言执行，速度较慢。
    内存效率：numpy 数组在存储大数据集时比 Python 列表的内存效率更高。numpy 数组内的元素通常是同一类型的，允许数组在内存中紧密排列，而 Python 列表则可以包含不同类型的元素，需要更多的存储空间。
  2. 功能
    广播功能：numpy 提供了广播功能，这允许不同大小的数组在算术运算中进行兼容。这使得数据处理变得更加方便和快捷。
    复杂数学运算支持：numpy 提供了大量的数学函数库，可以直接在数组上进行操作，如线性代数运算、统计分析等。
  3. 多维数组
    支持多维数组：numpy 支持多维数组（矩阵），而 Python 列表仅支持一维数组，虽然可以通过列表的列表实现多维数组，但这样做既不方便也不高效。
  4. 索引和切片
    高级索引和切片：numpy 提供了更多高级的索引和切片方式，使得数据操作更加灵活和强大。

python数据类型

Python 是一种动态类型的编程语言，这意味着在编写代码时不需要显式声明变量的数据类型。Python 解释器会在运行时自动推断数据类型。Python 支持多种内置数据类型，这些类型可以分为几个主要类别：数字类型、序列类型、映射（字典）类型、集合类型、布尔类型以及 None 类型。

  1. 数字类型
    1.1 整数（int）：表示没有小数部分的数，可以是正数或负数。例如：3, -5, 42。
    1.2 浮点数（float）：表示有小数部分的数。例如：3.14, -0.001, 2.0。
    1.3 复数（complex）：包含实部和虚部，虚部以 j 或 J 表示。例如：2 + 3j。

  3. 布尔类型
    布尔（bool）：表示逻辑值 True 或 False。常用于条件判断和循环控制。

  4. None 类型
    None：特殊的数据类型，表示空值或无值，通常用于表示函数没有明确返回值。

Break和Continuous区别

break用于结束整个循环。
continuous不执行当前循环的statement，运行下一个循环。

return 和 yield的区别

return 关键字用于从函数中返回结果。当 return 被执行时，函数将立即终止，控制权和 return 后的值将返回给调用者。如果没有指定返回值，函数默认返回 None。

yield 关键字用于从一个函数返回一个值，并暂停执行，保存当前函数的状态，以便在下次从函数调用（或迭代）时从停止的地方继续执行。使用 yield 的函数被称为生成器。

生成器允许你按需生成值，而不是一次计算并返回所有值，这可以显著减少内存使用并提高计算效率。

python中的浅拷贝和深拷贝

这两种拷贝方式对于基本数据类型（如整数、浮点数和字符串）的影响不大，因为这些类型在赋值时自动创建新的实例。然而，对于复合数据类型（如列表、字典、集合等），选择正确的拷贝方式是非常重要的，因为它们可以包含其他对象的引用。

主要担心这些可变对象：list、dict、set。

浅拷贝创建一个新对象，但它包含的是对原始对象中包含的对象的引用（即地址），而不是对象的副本。这意味着，如果原始对象包含了对其他对象的引用，则新对象和原始对象将引用同一个子对象。

深拷贝创建一个新对象，并递归地复制原始对象中包含的所有对象。这意味着它不仅复制了原始对象，还复制了原始对象中的所有子对象（复制了对象的所有引用）。因此，原始对象和新对象不会共享任何子对象。

range 和 xrange的区别

在 Python 2 中，range 生成完整列表，而 xrange 生成一个惰性序列，类似于迭代器。在 Python 3中二者已合并range，实现的是 xrange

==判断两个对象的值是否相同

lambda 函数

在 Python 中，lambda 函数是一种小型的匿名函数。lambda 函数可以接收任意数量的参数，但只能有一个表达式。它们通常用于需要函数对象的地方，但又希望这个函数简单到足以用一行代码表示。

lambda 函数的基本语法如下：

lambda arguments: expression

arguments 是传递给 lambda 函数的参数，可以是多个，用逗号分隔。
expression 是一个关于参数的表达式，其结果就是 lambda 函数的返回值。

单引号、双引号、三引号区别

单引号双引号没有区别，都可以用来表示字符串；
三引号，字符串内可直接换行，且可包含注释。

python 中传入参数的注意事项：

  1. 参数类型
    位置参数：这是最常见的参数类型，调用函数时根据参数的位置来传递值。
    关键字参数：允许你通过名称来指定参数，这使得函数调用更加清晰明了。
    默认参数：在定义函数时可以为参数提供默认值。使用默认参数可以使函数调用更灵活，同时减少需要传递的参数数量。

  2. 不可变与可变参数
    不可变对象：如整数、字符串和元组。在函数内部修改这些参数不会影响原始对象。
    可变对象：如列表和字典。如果在函数内部修改这些参数，调用者会看到这些更改，因为这类参数是通过引用传递的。

  3. *args 和 **kwargs
    *args：允许你将不定数量的位置参数传递给函数。
    **kwargs：允许你将不定数量的关键字参数传递给函数。
    这两种机制提供了额外的灵活性，尤其是在处理不确定数量的输入参数时。

  4. 参数传递方式
    按值传递：传递的是值的拷贝，对于不可变类型，函数内的修改不会影响原始数据。
    按引用传递：传递的是对象的引用，对于可变类型，函数内的修改会影响到原始数 据。
    Python 中的参数传递实际上是“按对象引用传递”（pass-by-object-reference），这意味着函数接收的是对象引用的拷贝。

  5. 避免使用可变对象作为默认参数
    默认参数只在函数定义时计算一次，如果使用了可变对象作为默认参数，那么每次调用该函数时，如果改变了这个参数的值，这个默认参数的值就会改变，可能会导致意外的行为。

装饰器

在 Python 中，装饰器是一种非常强大的功能，它允许你在不修改原始函数代码的情况下增加函数的功能。装饰器本质上是一个函数，它接受一个函数作为参数并返回一个新的函数。装饰器可以用来增强函数的行为，实现例如权限检查、日志记录、设置函数执行时间等功能。

Python 允许你使用 @ 符号（称为“语法糖”）来应用装饰器，这使得装饰器的应用更为简洁和直观。

函数和变量的作用域

在Python中，函数或变量的作用域是一个重要概念，它定义了变量或函数名称在程序中可以被访问的区域。Python中的作用域主要分为四种类型：

局部作用域（Local Scope）:

局部作用域是指在函数内部定义的变量。这些变量只能在函数内部被访问，函数外部无法访问。当函数调用结束后，局部作用域中的变量也会被销毁。

全局作用域（Global Scope）:

全局作用域是在模块级别定义的变量。在整个文件的任何地方都可以访问这些变量，除非它们被局部作用域中的同名变量遮蔽。

嵌套作用域（Enclosing Scope）:

当一个函数被另一个函数包围时，内部函数可以访问外部函数的局部变量。这种作用域有时也被称为非局部作用域。

内置作用域（Built-in Scope）:

Python自带的内置作用域包含了许多内置函数和异常，这些都是全局可用的。例如，print(), len() 等。
print(max([1, 2, 3])) # 使用内置函数max，输出 3

关键字 global 和 nonlocal:

global 关键字用来在函数内部声明变量为全局变量。
nonlocal 关键字用来在嵌套函数中声明变量为非局部变量。

解释型语言和编译型语言

编译型语言是那些需要使用编译器将程序源代码转换成机器语言（通常是二进制格式的可执行文件）的语言。这个转换过程发生在程序运行之前。 

性能：编译后的代码直接由硬件执行，因此运行速度快。  
一次编译，多次执行：编译过程只需要进行一次，之后可以多次运行编译后的代码，无需再次编译。  
平台依赖性：编译出的可执行文件通常是特定于操作系统和硬件的。不同平台需要重新编译。

解释型语言是那些不需要事先编译成机器语言的程序。源代码在程序运行时由解释器逐行读取，解释并执行。

特点：

灵活性和便捷性：代码修改后无需重新编译，可以直接运行，便于调试和测试。  
跨平台性：只要平台上有对应的解释器，相同的源代码可以在不同的操作系统上运行。  
性能问题：由于代码是在运行时逐行解释的，执行效率通常低于编译型语言。

new 和 init

new 是一个静态方法，它负责创建一个新的实例，它的主要任务是返回该类的一个实例。

init 是一个实例方法，它用于初始化新创建的对象。
init 并不返回任何值（或者说应该返回 None），它的主要职责是设置对象的初始状态。

如果需要控制对象的创建过程（例如实现单例模式），可以使用 __new__。如果只需要初始化对象，使用 init 就足够了。

类中self的概念及其三种应用

在Python中，self 是一个约定俗成的参数名称，它用于引用类的当前实例对象。当定义类的方法时，self 是方法中的第一个参数。通过使用 self，可以在类的方法中访问或修改对象的属性和其他方法。这是面向对象编程的一个核心概念，它使得类的实例可以访问和控制其自身的数据和行为。

三种应用场景：

访问实例属性： self 用于在类的方法中访问实例属性。实例属性是在类的方法中使用 self 创建的属性，它们对每个类的实例是唯一的。

修改实例属性： self 还可以用来修改实例的属性。这允许对象在其生命周期内改变状态。

在这里，update_age 方法使用 self 来修改 age 属性。

调用其他方法： self 可以用来在一个实例内部调用其他的方法。这是组织复杂行为和代码复用的一种方式。
